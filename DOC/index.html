Feature Viewer

This javascript tool will allow you to display your sequence features

Simple

This is the classic display of a sequence.
Before displaying the sequence you can adjust some rendering options :

  • showAxis (boolean) : Display or not the axis text
  • showSequence (boolean) : Display or not the sequence
  • brushActive (boolean) : Allows the zoom
  • toolbar (boolean) : Add a toolbar allowing users to download SVG and show help
  • toolbarPosition (string) : Position of toolbar in justify-content property, ex. toolbarPosition: space-between
  • zoomMax (integer) : Choose the max zoom allowed (number of residues) - Default: 10, cannot be less than 10
  • showSubFeatures (integer) : Allows subfeatures visualization - Default: false
  • flagColor (string) : Color of flags (titles on the left), hex string ex. flagColor: '#DFD5F5'
  • flagTrack (integer or string) : Dimensions (in px) of the left-side area (flags)
  • showLinkTag (boolean) : Shows link buttons (on the right)
  • showDisorderContentTag (boolean) : Shows disorder content tag (on the right)
  • buttonTrack (integer or string) : Dimensions (in px) of the right-side area (disorder tag and buttons)
  • animation (boolean) : Allows animations in zoom and sufeatures opening
  • offset (object) : Specify an offset (offset.start should always be > 0), ex. offset: {start: 40, end:100}

Add features

You can add features using the addFeature function.
Also the features have rendering options:

  • type ("rect","path","curve","unique","circle") : The type of feature, for a specific rendering
  • data (array of objects) : An array of objects, each containing :
    x (int) :
    first position
    y (int) :
    last position OR value [for features of type "curve" (curve value) OR "circle" (circle diameter, 0 to 1) OR extra tooltip annotation for "unique"]
    className (string) :
    html class for extra styling (Optional)
    color (string) :
    a custom color (Optional)
    stroke (string) :
    a custom stroke color (Optional)
    opacity (string) :
    custom opacity, 0 to 1 (Optional)
    tooltip (string) :
    custom message for tooltip (Optional)
  • links (array of objects) : An array of objects, each containing :
    name (string) :
    The button name, used to identify click event
    message (string) :
    Message for tooltip
    html (html string) :
    Style your button! Accepts material icons
  • label (string) : The name of this feature, which will be display as a label on the Y-axis (flag)
  • className (string) : A class name, for further personal computing (Optional)
  • height (int) : Height of the curve feature (default: 3, which means three times the height of a rectangle feature) (Optional)
  • yLim (int) : Max value of the curve values (Optional)
  • color (string) : The color of the features (Optional)
  • stroke (string) : The stroke color of the features (Optional)
  • opacity (string) : custom opacity, 0 to 1 (Optional)
  • tooltip (string) : Message for flag tooltip (Optional)

Add subfeatures

Feature viewer listeners can have several usages and are listed below

  • onButtonSelected: A function that is triggered when the flag is selected.
  • onClearSelection: Triggered when the viewer is zoomed out.
  • onFeatureSelected: Triggered when a feature (ex. rectangle) is selected.
  • onFlagSelected: Triggered to open subfeature details.
  • onZoom: Triggered when viewer is zoomed.

Subfeatures are dinamically visualized when flag is selected, when onFlagSelected is triggered, through the addSubFeature function.
Each subfeature has the same structure of features, and is added with function with two parameters

  • parentId: output of onFlagSelected.detail.dom_id
  • subFeatures: list of subfeatures
